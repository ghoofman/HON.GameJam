==================
= Samurai Showdown
==================
	* Created for the Home of Nerds 2017 Game Jam
	* Best played with a gamepad


==================
= Controls
==================

 Keyboard
 -------------------
	Move : WASD
	Attack : X
	Gather : Y
	Sacrifice : B
	Run : Left Shift
	Accept : Enter

 Gamepad
 ------------------
	Move : Left Thumbstick
	Attack : X
	Gather : Y
	Sacrifice : B
	Run : Right Trigger
	Accept : A


==================
= Mechanics / Gameplay
==================

* Killing an enemy gives +1 honor
* Sacrificing a samurai gives +8 honor
* Sacrificing, kills the lowest health samurai 
* If a samurai dies, you get 0 honor
* Each level ends when all enemies have been killed
* If all of your samurai die, you lose
